## Overview

Manual resume screening is time-consuming and prone to human bias. This project automates the resume evaluation process using Natural Language Processing (NLP) and machine learning techniques to classify resumes into relevant job categories.

The system processes resume text, extracts meaningful features using TF-IDF, and applies a trained machine learning model to generate accurate predictions through a simple application interface.

---

## Key Features

- Automated resume screening using machine learning
- NLP-based text preprocessing and TF-IDF feature extraction
- End-to-end pipeline from data preprocessing to real-time prediction

---

## Tech Stack

- Programming Language: Python  
- Libraries: Pandas, NumPy, Scikit-learn, NLTK  
- Feature Extraction: TF-IDF  
- Model: Machine Learning Classifier  
- Interface: Python application (app.py)

---

## Project Workflow

1. Resume text extraction and cleaning  
2. Text preprocessing (tokenization, stopword removal)  
3. Feature extraction using TF-IDF  
4. Model training and evaluation  
5. Resume classification through application interface
